# Keyrus Test

## Prerequisites:
- [Node](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/en/)


## Run Project
- `yarn install`
- `yarn dev`

## Test
- `yarn test`

## ESLint
- `yarn lint`

## Prettier
- `yarn format`

## Production
- `yarn build`
